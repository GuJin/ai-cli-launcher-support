# AI CLI Launcher Support

This repository is the public issue tracker for AI CLI Launcher, a macOS external-terminal launcher for local AI CLI workflows in JetBrains IDEs.

Use Issues to report bugs, request terminal support, or suggest workflow improvements.

Before opening a bug report, please include as much of this information as possible:

- AI CLI Launcher version
- JetBrains IDE name and version
- macOS version
- Terminal app and version
- Configured command, such as `codex`, `claude`, or a custom wrapper script
- Whether the Settings `Test` action works
- The `Copy Debug Report` output when it is relevant and safe to share
